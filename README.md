# Notes

## Compiling the PDF

Should work with any LaTeX compiler, I used `tectonic`.

> On MacOS you can install tectonic VIA:
> ```
> $ brew install tectonic
> ```



```shell
$ ./scripts/build.sh
```

Results in `./output/main.pdf`.
